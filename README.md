**Deskripsi Singkat:**
Proyek aplikasi Sistem Manajemen Blog (CMS) menggunakan framework Laravel. Aplikasi ini mencakup halaman pengunjung publik untuk membaca dan memfilter artikel, serta halaman dashboard admin untuk mengelola (CRUD) artikel, kategori, dan penulis.

**Cara Menjalankan Aplikasi Lokal:**
1. Clone repositori ini.
2. Buka terminal dan jalankan `composer install`.
3. Salin file `.env.example` menjadi `.env` lalu sesuaikan konfigurasi database `db_blog`.
4. Jalankan `php artisan key:generate`.
5. Jalankan `php artisan storage:link`.
6. Jalankan `php artisan serve`.
